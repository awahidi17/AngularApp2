# AngularApp2 Task Tracker

A task tracker built with Angular standalone components, a shared service, custom pipes, and a custom directive.

## Features

- Add a task with title, description, priority, and due date
- Mark tasks as completed
- Filter by all, active, completed, or overdue
- Clear completed tasks
- Live stats powered by RxJS observables
- Custom pipe for due date labels
- Custom pipe for long description trimming
- Custom directive for task card styling

## Run the project

```bash
npm install
ng serve --open
```

## Main Angular topics used

- Standalone components
- Reactive forms
- Dependency injection
- RxJS BehaviorSubject and observables
- Custom pipes
- Custom directive
